Configuration du Tunnel Local (LocalTunnel)

📋 Prérequis
Node.js installé (v14 ou supérieur recommandé).
Un serveur tournant localement sur le port 8000.
Connaître son adresse IP publique (pour les règles de firewall si nécessaire).

🚀 Installation
npm install -g localtunnel

⚙️ Vérification de l'IP Publique
curl icanhazip.com

▶️ Lancer le Tunnel
lt --port 8000 --subdomain feosync
